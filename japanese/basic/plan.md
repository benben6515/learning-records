# 五十音練習資源包

## 一、記憶口訣（平假名優先）

### あ行（母音）- 超重要

```
あ (a) - 「安」字草書，張嘴說啊
い (i) - 「以」字簡化，一個人站著
う (u) - 像兔子耳朵，發「嗚」聲
え (e) - 「衣」服的樣子
お (o) - 「於」字變形，嘴巴圓圓
```

### か行（ka行）

```
か (ka) - 「加」法符號的樣子
き (ki) - 鑰匙 (key) 的形狀
く (ku) - 嘴巴 (くち) 的「く」
け (ke) - 「計」算的草書
こ (ko) - 兩條橫線，像「二」
```

### さ行（sa行）

```
さ (sa) - 像「左」字的簡化
し (shi) - 微笑 (smile) 的嘴型
す (su) - 像「寸」字
せ (se) - 「世」界的草書
そ (so) - 縫紉 (sew) 的針線
```

### た行（ta行）

```
た (ta) - 「太」陽的草書
ち (chi) - 像數字「5」
つ (tsu) - 笑臉符號 :)
て (te) - 「天」空的樣子
と (to) - 「止」步的符號
```

### な行（na行）

```
な (na) - 「奈」良的草書
に (ni) - 「二」加一撇
ぬ (nu) - 「奴」隸的樣子
ね (ne) - 「祢」的簡化，有個圈
の (no) - NO! 的禁止符號
```

### は行（ha行）

```
は (ha) - 「波」浪兩個點
ひ (hi) - 笑臉 hi!
ふ (fu) - 富士山 (Fuji)
へ (he) - 房子的屋頂
ほ (ho) - 「保」存，有個圈
```

### ま行（ma行）

```
ま (ma) - 媽媽 (mama) 抱小孩
み (mi) - 數字「3」彎曲
む (mu) - 牛 (moo) 的叫聲
め (me) - 「女」字加一筆
も (mo) - 「毛」筆的樣子
```

### や行（ya行）

```
や (ya) - 「也」字的草書
ゆ (yu) - 「由」來的樣子
よ (yo) - 「与」的簡化
```

### ら行（ra行）

```
ら (ra) - 「良」的草書
り (ri) - 「利」刀的形狀
る (ru) - 路 (route) 的彎曲
れ (re) - 「禮」物的樣子
ろ (ro) - 「呂」字的變形
```

### わ行 + ん

```
わ (wa) - 「和」平的草書
を (wo) - 只用於助詞，像數字「5」
ん (n) - 鼻音，像「ん～」的樣子
```

## 二、容易混淆的字 - 對照記憶

### 平假名混淆組

| 組別 | 字             | 區別口訣                                 |
| ---- | -------------- | ---------------------------------------- |
| 1    | ぬ vs ね       | **ぬ**是奴隸低頭，**ね**是貓咪有圈圈耳朵 |
| 2    | め vs ぬ       | **め**像女字加一撇，**ぬ**有個大圈圈     |
| 3    | は vs ほ       | **は**兩撇沒圈，**ほ**有圈像保險箱       |
| 4    | き vs さ       | **き**是直的鑰匙，**さ**是彎的左邊       |
| 5    | る vs ろ       | **る**一筆完成路線，**ろ**兩筆像呂布     |
| 6    | わ vs ね vs れ | **わ**圈在上，**ね**圈在右下，**れ**沒圈 |

### 片假名混淆組（之後學）

| 組別 | 字       | 區別口訣                               |
| ---- | -------- | -------------------------------------- |
| 1    | シ vs ツ | **シ**直立像smile，**ツ**躺著像tsunami |
| 2    | ソ vs ン | **ソ**往左撇，**ン**往右撇             |
| 3    | ク vs ケ | **ク**兩筆，**ケ**三筆                 |

## 六、自我測驗方法

### 測驗1：認讀測驗（第1週末）

```
看到假名能立刻說出發音：
あ → (a)   か → (ka)   さ → (sa)
い → (i)   き → (ki)   し → (shi)
（隨機排列46個字，能認出40個以上就及格）
```

### 測驗2：聽寫測驗（第2週末）

```
聽到發音能立刻寫出假名：
(a) → あ    (ka) → か    (sa) → さ
(i) → い    (ki) → き    (shi) → し
（用 YouTube 五十音歌測試）
```

### 測驗3：單字測驗（第3週）

```
看懂簡單單字：
ねこ → neko → 貓
いぬ → inu → 狗
さくら → sakura → 櫻花
```
