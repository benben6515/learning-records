# 五十音學習策略

## 一、先理解平假名與片假名的用途

### 平假名（ひらがな）

- 用於：日文原生詞彙、文法助詞、動詞變化
- **先學這個！** 因為最常用

### 片假名（カタカナ）

- 用於：外來語、擬聲詞、強調
- 平假名熟練後再學

## 二、分段學習法（推薦！）

## 三、有效的記憶技巧

### 1. **聯想記憶法**

為每個假名創造故事：

**平假名例子：**

- **あ**：像一個人張開嘴說「啊～」
- **き**：像一把「key」(鑰匙)
- **の**：像「No」的符號

**片假名例子：**

- **カ**：像刀「cut」的形狀
- **ト**：像「ten」的 T
- **シ**：像笑臉 smile 的 si

### 2. **筆順練習法**

- 用田字格練習本，每個字寫10-20遍
- **筆順很重要！** 正確筆順能幫助記憶

### 3. **對比記憶法**

容易混淆的字一起記：

| 平假名   | 片假名   | 區別方法                               |
| -------- | -------- | -------------------------------------- |
| ぬ vs ね | ヌ vs ネ | ぬ像「奴」，ね像「根」/ ヌ往右，ネ往左 |
| は vs ほ | ハ vs ホ | は兩撇，ほ有圈 / ハ兩撇，ホ三筆        |
| シ vs ツ | -        | シ直立，ツ斜躺                         |
| ソ vs ン | -        | ソ往左，ン往右                         |

### 4. **單字記憶法**

不要死背字母，用簡單單字來記：

| 單字 | 平假名 | 意思          |
| ---- | ------ | ------------- |
| 愛   | あい   | ai (愛)       |
| 海   | うみ   | umi (海)      |
| 猫   | ねこ   | neko (貓)     |
| 犬   | いぬ   | inu (狗)      |
| 桜   | さくら | sakura (櫻花) |

## 四、實戰練習工具

### 線上資源

1. **Duolingo** - 遊戲化學習
2. **Anki** - 間隔重複記憶卡
3. **Write It! Japanese** - 筆順練習 app
4. **Tofugu** - 記憶法文章

### 每日練習計劃

- **早上 (10分鐘)**：寫5-10個新字，每個寫5遍
- **中午 (5分鐘)**：用 app 複習
- **晚上 (10分鐘)**：默寫今天學的字，測試自己

## 五、常見錯誤避免

❌ **不要一次學太多**：一天最多學5-10個
❌ **不要只看不寫**：一定要動手寫
❌ **不要平片假名一起學**：先精通平假名
❌ **不要跳過複習**：前一天的字要每天複習

## 六、進階：濁音・半濁音

掌握基本五十音後再學這些：

### 濁音（加兩點 ゛）

| 原字   | 濁音   | 發音變化 |
| ------ | ------ | -------- |
| か(ka) | が(ga) | k → g    |
| さ(sa) | ざ(za) | s → z    |
| た(ta) | だ(da) | t → d    |
| は(ha) | ば(ba) | h → b    |

### 半濁音（加圓圈 ゜）

| 原字   | 半濁音 | 發音變化 |
| ------ | ------ | -------- |
| は(ha) | ぱ(pa) | h → p    |
| ひ(hi) | ぴ(pi) | h → p    |
| ふ(fu) | ぷ(pu) | h → p    |

## 七、3週速成計劃

**第1週**：清音（あ行～わ行）+ 每天複習
**第2週**：濁音、半濁音 + 綜合測驗
**第3週**：拗音（きゃ、きゅ、きょ等）+ 實際閱讀練習

## 實用測試方法

1. **隨機測試**：把字卡打亂，看能認出多少
2. **聽寫測試**：用 YouTube 五十音歌，邊聽邊寫
3. **實戰閱讀**：試著讀日文兒童繪本
